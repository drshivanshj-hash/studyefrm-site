# STUDYEFRM

Structured reproductive medicine learning and clinical reasoning platform for EFRM preparation, embryology workflows, OSCE systems, and evidence-based fertility education.

## Current Stack
- Static frontend
- Netlify Functions
- Claude API integration
- Netlify Blobs persistence

## Core Features
- Case ingestion
- AI-assisted reasoning
- Moderation workflow
- Teaching card publication
- OSCE-oriented educational transformation

## Environment Variables
- ANTHROPIC_API_KEY
- ADMIN_TOKEN
